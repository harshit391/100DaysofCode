# 100 Days of Code - DSA in Java

A structured journey through **Java fundamentals and Data Structures & Algorithms**, solving **230+ LeetCode problems** over 99 days. The first 12 days focus on core Java concepts — JVM, JDK, OOP principles, classes, objects, inheritance, polymorphism, and more — building a strong foundation before diving into problem-solving from Day 13 onward.

## Overview

| Metric | Detail |
|--------|--------|
| **Language** | Java |
| **Problems Solved** | 230+ |
| **Days Tracked** | 99 (Day 1 - Day 99) |
| **Source** | LeetCode (Day 13+) |
| **Difficulty Range** | Fundamentals to Hard |

## Topics Covered

- **Java Core** - JVM, JDK, JRE, data types, variables, operators, control flow, loops
- **OOP Concepts** - Classes, objects, constructors, inheritance, polymorphism, abstraction, encapsulation
- **Strings** - Reversal, anagrams, pattern matching, compression, palindromes
- **Arrays** - Two pointers, prefix sum, matrix operations, spiral traversal
- **Sorting** - Bubble sort, insertion sort, selection sort, merge sort, quick sort, cyclic sort
- **Binary Search** - Rotated arrays, 2D matrices, mountain arrays, infinite arrays, floor/ceiling
- **Recursion** - Power functions, Fibonacci, binary search, Tower of Hanoi
- **Sliding Window** - Subarray products, longest substring, character replacement
- **Stack** - Valid parentheses, reverse polish notation, basic calculator, asteroid collision, largest rectangle in histogram
- **Dynamic Programming** - Longest valid parentheses, maximum product subarray
- **Math & Geometry** - Roman numeral conversion, prime counting, points on a line, triangle checks
- **Greedy** - Candy distribution, lemonade change, assign cookies
- **Intervals** - Merge intervals, summary ranges

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/HarshitSingla/100DaysofCode.git
   ```
2. Navigate to any day folder:
   ```bash
   cd Day13
   ```
3. Compile and run a solution:
   ```bash
   javac InttoRoman.java
   java Day13.InttoRoman
   ```
